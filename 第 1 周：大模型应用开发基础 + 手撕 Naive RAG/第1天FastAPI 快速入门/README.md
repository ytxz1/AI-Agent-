# 第 1 天：FastAPI 快速入门

今天的学习主题是 FastAPI 快速入门。目标是掌握 FastAPI 基础，能够创建路由、处理请求，并为后续 LangChain 和 Naive RAG 项目提供 API 服务基础。

## 今日完成标准

学完今天内容后，你应该能够：

1. 创建一个最小 FastAPI 应用。
2. 使用 Uvicorn 启动服务。
3. 编写 GET、POST、PUT、DELETE 路由。
4. 使用路径参数处理资源 ID。
5. 使用查询参数处理分页、搜索、开关。
6. 使用 Pydantic 模型接收 JSON 请求体。
7. 使用 `/docs` 测试接口。
8. 解释 FastAPI 自动校验和自动文档的价值。
